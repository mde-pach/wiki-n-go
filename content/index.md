# Welcome to the Wiki

This page is **served from GitHub via jsDelivr** and rendered in your browser at
runtime. Editing this file and committing it updates the page **with no
rebuild** — the reader always fetches the latest commit.

## How it works

- Content lives as Markdown in a GitHub repo (the database).
- The site (this interface) fetches it from the CDN, pinned to the latest commit
  SHA, and renders it client-side.
- Soon: edit **in this site** with Wikipedia-level friction — no account, no
  token.
