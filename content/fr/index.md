# Bienvenue sur le wiki

Cette page est **servie depuis GitHub via jsDelivr** et rendue dans votre
navigateur à l'exécution. Modifier ce fichier et le valider met à jour la page
**sans reconstruction** — le lecteur récupère toujours le dernier commit.

## Comment ça marche ?

- Le contenu vit en Markdown dans un dépôt GitHub (la base de données).
- Le site récupère le contenu depuis le [[w:Réseau de diffusion de contenu|CDN]],
  épinglé au dernier commit, et le rend côté client.
