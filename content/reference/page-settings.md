# Page settings

When you [[help/editing|edit]] a page, a **Properties** panel holds that page's own options. You fill in a form — there's nothing to hand-write — and each option maps to a visible part of the page, most of which are pictured on [[anatomy-of-a-page|Anatomy of a page]].

| Setting | What it does |
|---|---|
| **Summary** | A one-line description, shown in search results and hover previews. |
| **Top note** | A short italic note at the very top — usually a pointer to a related page (a *hatnote*). |
| **Notice** | A coloured banner across the top, info or warning, for drafts and cautions. |
| **Label** | The small word above the title, like "Concepts" or "Guide". |
| **Header image** | A picture shown with the page and used when the page is shared. |
| **Quick facts** | The boxed table of facts beside the opening — the *infobox*. |
| **Categories** | The topics this page belongs to (see [[categories|Categories]]). |
| **Who can edit** | The page's [[trust-and-moderation|protection level]] — open, auto, extended, or maintainer. |
| **Page look** | Override the default style, light/dark, width, or text size for just this page. |
| **Redirect to** | Send anyone who opens this page straight to another one. |
| **Translation group** | Links this page to its versions in other languages (see [[multilingual|Multilingual wikis]]). |

You won't touch most of these on most pages. A summary, and maybe a category, is plenty for an ordinary article.
