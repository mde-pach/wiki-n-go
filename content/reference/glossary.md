# Glossary

Short definitions of the words this wiki uses, with a link to the fuller page for each. If a term sends you somewhere unfamiliar, this is the place to start.

**Anonymous editing** — Editing without an account, credited to a derived nickname rather than a name. See [[anonymous-editing|Anonymous editing]].

**Backlink** — The set of pages that link *to* a given page; the reverse of an ordinary link. See [[links-and-references|Links and references]].

**Category** — A topic that pages are filed under, so the wiki can be browsed by subject. See [[categories|Categories]].

**CDN** — A [[w:Content delivery network|content delivery network]]: the global file host that serves Wikigit pages for reading. See [[how-it-works|Architecture]].

**Engine** — The small [Bun](https://bun.sh/) server that saves edits and serves the dynamic parts of a wiki — the only custom infrastructure Wikigit runs. See [[how-it-works|Architecture]].

**Hatnote** — The short italic pointer above a page's first line, used for "for X, see Y." See [[anatomy-of-a-page|Anatomy of a page]].

**Infobox** — The boxed table of facts beside a page's opening. See [[anatomy-of-a-page|Anatomy of a page]].

**Interwiki link** — A link out to Wikipedia, written with a `w:` prefix. See [[links-and-references|Links and references]].

**Maintenance category** — A housekeeping label like "draft" or "needs sources," kept apart from real topic categories. See [[categories|Categories]].

**Markdown** — The plain-text formatting Wikigit pages are written in. See [[help/formatting|Formatting]].

**Protection** — A page's edit restriction: open, auto, extended, or maintainer. See [[trust-and-moderation|Trust and moderation]].

**Red link** — A link to a page that doesn't exist yet, shown in red, that offers to create it. See [[links-and-references|Links and references]].

**Redirect** — A page that forwards a reader to another, left behind automatically when a page is renamed. See [[help/organizing|Organizing]].

**Revision** — One saved version of a page; the history is the page's [[w:Git|Git]] commits. See [[revisions|Revision history]].

**Special page** — An auto-generated report about the wiki, like recent changes or wanted pages. See [[special-pages|Special pages]].

**Talk page** — The discussion area attached to every page. See [[talk-pages|Talk pages]].

**Transclusion** — Pulling one page's content into another with `{{name}}`. See [[help/formatting|Formatting]].

**Translation group** — The shared label that ties an article's versions in different languages together. See [[multilingual|Multilingual wikis]].

**Trust tier** — A contributor's standing, earned through accepted edits, that decides whether their changes publish on their own. See [[trust-and-moderation|Trust and moderation]].

**Wiki** — A website a group writes together, one linked page at a time. See [[index|Wikigit]].

**Wikilink** — A link between pages, written in `[[double brackets]]`. See [[links-and-references|Links and references]].
