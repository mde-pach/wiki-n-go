# Help

Welcome — this page explains how the wiki works and points you to the rest of
the help. You don't need an account to read or to edit.

## How it works

Every page is Markdown stored in a GitHub repository. The site fetches the
latest committed version from a CDN and renders it in your browser, so an edit
goes live **without rebuilding the site**. The repository's history *is* the
page history — every revision is a commit.

You stay on this site the whole time: there is no "edit on GitHub" hand-off and
no token to paste. A small relay turns your edit into a commit for you.

## Who can edit

Anyone. Edits are signed automatically — either as your GitHub identity if you
chose to sign in, or as an anonymous pseudonym derived from your network. We
never store a raw IP or email. See [[help/editing|Editing and contributing]]
for the details.
