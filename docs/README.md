# Wikigit

Wikigit is a wiki you can set up in a few minutes and let anyone help write. It's free to run, your pages stay yours, and people can edit right in their browser — no sign-up required.

New here? Start with **[What is Wikigit?](what-is-wikigit.md)**, then **[Create your wiki](create-your-wiki.md)**.

- **[Make it yours](customize.md)** — name, logo, colors, home page, address, and languages.
